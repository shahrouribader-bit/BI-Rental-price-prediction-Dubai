# 🛠️ Tools Research and Selection

## Tools Evaluated

| Tool | Decision | Reason |

|---|---|---|

| **Python 3** | ✅ Selected | Best ML pipeline support: pandas, scikit-learn, matplotlib, seaborn |

| R | ❌ Rejected | Less ML pipeline support |

| **Microsoft Power BI Desktop** | ✅ Selected | Free, DAX measures, Key Influencers AI visual, interactive slicers |

| Tableau | ❌ Rejected | Enterprise licensing incompatible with academic budget |

| Looker | ❌ Rejected | Cloud only |

| Excel | ❌ Rejected | Cannot scale to 263,000+ rows |

| **Jupyter Notebook** | ✅ Selected | Reproducible documentation for academic submission |

## Technology Stack

Data Source → Processing → ML Engine → Export → Dashboard

DLD Open Data → Python + Pandas → Scikit-Learn → CSV → Power BI

## Key Libraries

- pandas — data processing

- scikit-learn — ML models and pipeline

- matplotlib + seaborn — visualizations

- joblib — model saving

- numpy — numerical operations
