# 🎯 Business Intelligence Project Description and Objectives
## What is this project about?
This project develops a data-driven rental price prediction system for the UAE real estate market using machine learning and Business Intelligence tools. It addresses pricing opacity in Dubai's rental market where agents quote dramatically different prices for similar properties with no objective benchmark available.
The median value of AED 65,268 is adopted as the market benchmark, as the mean of AED 321,000 is influenced by high-value commercial contracts. Dubai's market has a highly positively skewed distribution making median the appropriate measure.
## 🎯 Project Objectives
1. 🔮 Predict the total rental CONTRACT_AMOUNT for any property given its characteristics
2. 🔍 Identify which factors most strongly influence rental price
3. 📊 Flag undervalued and overvalued properties using GROWTH_PERCENTAGE
4. 📅 Forecast future rental prices: Next Month, Next Quarter, Next Year
5. 📱 Build a three-page Microsoft Power BI interactive dashboard
6. 💡 Provide investment recommendations based on area-level growth percentage analysis
## ❓ Business Questions Answered
- What factors affect property rental prices in the UAE?
- Which areas and metro zones have the highest rental values?
- What is the expected future rental price for a given property profile?
- Which properties represent the best investment opportunities?
- Where are properties currently overvalued or undervalued?
