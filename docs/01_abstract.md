# 📄 Abstract
The project presents an end-to-end Business Intelligence solution for predicting rental contract amounts in the UAE real estate market. The UAE real estate market is one of the fastest growing globally, creating an urgent need for a data-driven pricing tool that provides objective benchmarks for tenants, investors, and policymakers. The project analyzes more than 300,000 registered contracts from the Dubai Land Department.
The raw data was preprocessed in seven stages of data cleansing with four engineered features applied to ensure data quality. Thirteen predictive features were selected to train and compare five machine learning models: Linear Regression, Random Forest, Ridge, Decision Tree, and Neural Network. The best result was achieved by Random Forest with R²=0.9510 and MAE=AED 71,130. All outputs were exported as CSV and integrated into a three-page interactive Power BI dashboard.
Key insights show that contract duration (r=0.56), building scale (r=0.53), and metro location are the major determinants of rental prices. Al Garhoud has the highest growth percentage making it ideal for investment. Linear models reached only 52% while Random Forest achieved 95.10%, confirming that Dubai rental pricing is fundamentally non-linear.
**🔑 Keywords:** Real estate price prediction, machine learning regression, Random Forest, Business Intelligence, UAE property market, Power BI, Dashboard.
